# What Is Rune?

## Short answer

Rune is a toolkit for building animated ASCII visuals in React apps.

ASCII means text characters like `#`, `.`, `@`, and `*`.
Rune treats those characters like visual building blocks, similar to pixels.

## Why Rune exists

Most visual animation libraries are focused on SVG, Canvas graphics, or video.
Rune is for people who want a text-art style that still feels modern and reusable.

The goal is:

- easy to drop into a React app
- lightweight by default
- theme-aware (light/dark mode)
- composable like UI components
